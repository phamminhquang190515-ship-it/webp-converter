## 🔧 Installation Steps

1. **Download the Application**: Use the link provided above to visit the download page. 
2. **Extract the Files**: Once downloaded, extract the files to a location on your computer.
3. **Open a Terminal or Command Prompt**:
   - On Windows: Press `Win + R`, type `cmd`, and press `Enter`.
   - On macOS: Open `Terminal` from your Applications.
   - On Linux: Open your preferred terminal emulator.
4. **Navigate to the Folder**: Change your directory to where you extracted the files. Use:
   ```
   cd path_to_your_downloaded_folder
   ```
5. **Install the Required Packages**: Run the following command to install the required libraries:
   ```
   pip install -r requirements.txt
   ```
6. **Run the Application**: After installation, start the webp-converter by executing:
   ```
   python main.py
   ```

## 🖼️ How to Use

1. **Select Your Image**: Click on the "Choose File" button to select the image you want to convert.
2. **Choose Desired Settings**: Set the WebP quality and other options.
3. **Convert the Image**: Click the "Convert" button to start the conversion.
4. **Download the WebP File**: Once the conversion is finished, a download link will appear for your newly created WebP image.

## 🛠️ Troubleshooting

If you encounter any issues while using webp-converter, consider the following:

- Ensure you have Python installed. If not, download it from [Python’s official site](https://www.python.org/downloads/).
- Check that the required packages are installed correctly by running `pip install -r requirements.txt` again.
- Make sure the file you are trying to convert is not too large. Try a smaller image if you have problems.
